# Routes

Framework: React + Vite single page app.

## `/`

- URL path: `/`
- Entry: `src/main.tsx`
- Root component: `src/App.tsx`
- Layout: inline in `App`
- Summary: Passwordless name selection, then the Bratislava trip board with tabs for Reiseplan, Ideen, Bierliste, Kosten, and Teilnehmer.
